## Quick Overview

**What You'll Need:**
- ESP32 development board
- Pro-Lite ASCII-Series display
- 3 wires for serial connection
- USB cable for programming
- Power supply

**Basic Steps:**
1. Flash firmware to ESP32
2. Wire ESP32 to Pro-Lite display
3. Configure WiFi
4. Test and enjoy!
